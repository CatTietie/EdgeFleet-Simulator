# EdgeFleet Simulator

## 概述
EdgeFleet Simulator 是一个物联网设备模拟与规则引擎平台，允许运维团队演练大规模设备接入场景。你可以快速生成数千个虚拟传感器，通过 MQTT 上报温湿度等数据，在实时仪表盘上监控设备状态和历史曲线，并自由组合条件设置告警规则。不同部门之间的数据与告警完全隔离，适合内部培训和生产前压力测试。

## 核心功能
- **设备模拟器**：按模板批量创建虚拟设备，自定义上报频率与心跳。
- **MQTT 消息网关**：统一接入设备数据，支持上下线状态管理。
- **实时仪表盘**：设备拓扑图与历史曲线，数据自动刷新。
- **告警规则引擎**：支持多条件组合（大于/小于/连续超限）、告警/恢复双状态通知及 Webhook 推送。
- **多租户权限**：组织-分组-用户三级隔离，保障数据安全。

## 技术栈
- 后端：Python / FastAPI, MQTT Broker (Mosquitto), InfluxDB
- 前端：React, ECharts, D3.js
- 通信：WebSocket, MQTT
- 部署：Docker Compose

## 快速开始
1. 克隆仓库并进入目录。
2. 启动服务：`docker-compose up -d`
3. 访问前端：`http://localhost:3000`
4. 阅读 `docs/` 下的协议规范，了解 MQTT 主题格式、告警规则 DSL 和权限模型。

## 测试
项目包含完整的测试套件：模拟 2000 设备并发上报、告警边界验证、权限隔离测试。运行 `pytest` 即可执行。
